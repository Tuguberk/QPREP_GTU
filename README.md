### Seçenek 2: Yerel Kurulum

```bash

git clone https://github.com/tuguberk/QPREP.git
cd QPREP

# Sanal ortam oluşturun (önerilen)
python -m venv venv
source venv/bin/activate  # macOS/Linux

pip install -r requirements.txt
```
